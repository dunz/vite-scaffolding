## Stack

- 번들러: `Vite`
- SPA 프레임웍: `React`
- 언어: `Typescript`
- 코드검사: `eslint`
- 코드검사 플러그인:
  - `@typescript-eslint`
  - `eslint-plugin-react`
  - `eslint-config-prettier`
- 코드스타일: `prettier`
